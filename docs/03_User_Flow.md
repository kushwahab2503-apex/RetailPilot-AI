# 1. Purpose

This document defines how users interact with RetailPilot AI from the moment they visit the website until they complete their business analysis.

The objective is to create a smooth, intuitive, and professional user experience.

---

# 2. User Types

## Visitor

A person visiting the website for the first time.

Capabilities:

* View Landing Page
* Read Features
* View About Section
* Try Demo
* Sign Up
* Login

---

## Registered User

A retailer using RetailPilot AI.

Capabilities:

* Upload Dataset
* Analyze Sales
* View Dashboard
* Train Models
* Generate Reports
* View AI Insights

# 18. User Experience Principles

RetailPilot AI should always provide:

* Simple Navigation
* Minimal Clicks
* Fast Response
* Modern UI
* Clear Data Visualization
* Actionable Business Insights

Every screen should help users make business decisions instead of simply displaying data.
